# Finite Privacy Policy

Last updated: 2026-06-02

Finite is a local-first app for intentional social media catch-up.

## Data Finite Stores

Finite stores the following data on the user's device:

- creator display names
- creator platforms
- creator profile URLs
- optional creator handles
- creator active/inactive status
- check frequency
- check-in history

## Data Finite Does Not Collect

Finite does not collect, transmit, sell, or share user data.

Finite does not use:

- user accounts
- cloud sync
- analytics
- advertising
- tracking pixels
- platform APIs
- scraping
- recommendation systems
- payment systems

## Backups

Finite can export local app data as JSON.

The exported backup may contain creator names, handles, profile URLs, and check-in history. Anyone with access to the backup file or pasted backup text can read that information.

Users are responsible for storing exported backups safely.

## External Links

Finite opens creator profile URLs in external apps or browsers.

Once a user opens an external platform, that platform's own privacy policy and tracking practices apply. Finite does not control external platforms.

## Data Deletion

Users can clear local Finite data from the Settings screen.

Deleting local data removes saved creators and check-ins from the device storage used by Finite.
