# Play Store Listing Draft

## App Name

Finite

## Short Description

Keep up with selected creators, then leave.

## Full Description

Finite helps you use social media intentionally.

Instead of opening an infinite feed, add the creators, channels, accounts, or profiles you actually want to check. Finite creates a finite catch-up queue, opens each profile in the original platform or browser, and lets you mark each one done or skipped.

When the queue ends, you are done.

Finite is not a social network, feed, blocker, or recommendation app. It is a local-first routing layer for deliberate social media use.

Features:

- add creators manually
- choose Instagram, TikTok, YouTube, X, Reddit, or Other
- set check frequency
- pause creators without deleting them
- open creator profile URLs externally
- swipe through creators as done or skipped
- view recent check-ins
- export and import a local JSON backup
- clear local data

Finite does not use accounts, cloud sync, analytics, ads, scraping, or platform APIs.

## Suggested Category

Productivity

## Content Notes

Finite does not display user-generated social media content inside the app. It only stores user-entered creator/profile links and opens them externally.

## Data Safety Draft

Data collected: none by the developer.

Data shared: none.

Local data: creator names, URLs, handles, settings, and check-ins are stored on the user's device.

Data deletion: users can clear local data in Settings.
